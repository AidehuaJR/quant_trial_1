# Dehua AI

A Korean-market autonomous trading dashboard prototype focused on transparent paper trading, interactive chart analysis, and future Toss Securities integration.

## Current prototype

- Searchable Korean representative stocks
- Interactive candlestick and volume charts
- 1 day, 3 day, 1 week, 1 month, 3 month, and 1 year ranges
- Zoom, drag, crosshair OHLCV details, refresh, and last-updated status
- User-defined buy, stop-loss, and take-profit levels
- Paper-trading and simulated-price labels

> All displayed prices and activity are simulated until an authorized broker market-data feed is connected. This software is not financial advice.

## Development

```bash
npm install
npm run dev
```
